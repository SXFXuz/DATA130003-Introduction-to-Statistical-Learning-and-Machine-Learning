5.建立以月租金（rent）为因变量，其余为自变量的线性回归模型

建立回归模型。

定性变量的基准组为：次卧(租赁房间)、低楼层(所在楼层)、否(邻近地铁)、石景山(城区)、自采暖(供暖方式)

data2 <- data
data2$room <- relevel(data2$room,'次卧')
data2$floor_grp <- relevel(data2$floor_grp,'低楼层')
data2$subway <- relevel(data2$subway,'否')
data2$region <- relevel(data2$region,'石景山')
data2$heating <- relevel(data2$heating,'自采暖')
#改变因子水平次序，使其与题目要求一致。

lin <- lm(rent~area + room + bedroom + livingroom + bathroom + heating + floor_grp + subway + region,data = data2)

模型中回归系数表示在其他自变量一定的情况下，对应自变量增加一单位因变量的变化量；或对应自变量取值为不同类别时，相对于基准组因变量的变化量。以下对某回归系数的说明均是在其他自变量一定的情况下。

城区以石景山为基准，昌平区平均月租金比石景山高57.2元，朝阳区平均月租金比石景山高631.7元，大兴区平均月租金比石景山低422.0元。其他城区以此类推。 城区回归系数由大到小依次为：西城、海淀、朝阳、东城、丰台、昌平、石景山(基准)、通州、大兴、顺义、房山。该顺序与各城区按月租金平均值或中位数排序大致相同，不同处只有丰台、昌平两城区排在了石景山前面。通过柱状图和箱线图可以看出，这三组月租金平均值(或中位数)相近，但丰台、昌平的数据更加分散，可能是城区与面积的交互作用导致了相应回归系数的差异。

有地铁的房源平均月租金较无地铁的高约280元。集中供暖的房源平均月租金比自供暖的房源高约155.8元。有主卧的房源平均月租金较仅有次卧的房源高约0.08元。这说明有集中供暖、附近有地铁的房源更受租房者欢迎，而租房者几乎不关心房间是否有主卧。

中楼层、高楼层的房源平均月租金较低楼层的房源分别低约55.6元、25.0元。这说明租房者不太关心房源所处的楼层,相对更偏爱低层。

卧室每增加一间，平均月租金减少约90.6元。卫生间每增加一间，平均月租金增加182.1元。客厅每增加一间，平均月租金减少约182.1元。房间面积每增加1平方米，平均月租金增加约76.7元。一般情况下，月租金随房间面积的增大而增大。房源客厅、卧室的数量增加会导致平均月租金减少，这可能是由于入住人数增加，每个人分担的租金减少；也可能是因为租房者由于自身经济能力问题不愿为更多的客厅、卧室承担更多的租金，导致这类房源需求量较小。

F统计量的p值很小，说明是显著的，至少有一个自变量对因变量有显著性影响。主卧、高楼层、昌平城区对应的p值较大，不能拒绝相应回归系数为0的原假设(显著性水平为5%下)，其与月租金的线性相关关系不显著。

其他变量对月租金的线性相关性是显著的(显著性水平为5%下)。

7.对 BIC 选择后的模型进行五折交叉验证，评估模型结果，并进行解读。

#首先编写用于k折交叉验证的函数
pred.cv <- function(dat,k)
{
  ind = sample(1:k,nrow(dat),replace = T)
  pred_cv = rep(0,nrow(dat))
  for (i in 1:k)
  {
    ii = which(ind == i)
    obj = lm(rent~area + bedroom + bathroom + heating + subway + region,data = data2[-ii,])
    pred_cv[ii] = predict(obj,dat[ii,])
  }
  rmse = sqrt(mean((pred_cv - data2$rent)^2))
  return(list(pred_cv = pred_cv,rmse = rmse))
}

#进行五十次五折交叉验证
set.seed(4321)
rmses = rep(0,50)
for (i in 1:50)
{
  pred_cv = pred.cv(dat = data2,k = 5)
  rmses[i] = pred_cv$rmse
}

#计算平均均方根误差
print(mean(rmses))
## [1] 455.5204

五折交叉验证的平均均方根误差约为455，占租金数据的平均值(2797.5)达约16%，误差相对较大。考虑到BIC变量选择后的模型调整后的R方为0.644，说明该模型对平均月租金有一定解释能力，但仍有改进的空间。预测不够准确的原因可能是模型有异方差现象，应该对因变量进行对数或平方根的变换；也可能是模型有非线性的问题，应该对自变量进行平方、对数或平方根的变换。
